# User Search Cards UI


<h3>LIVE DEMO :- </h3>
A clean and responsive profile card search application built using **HTML**, **CSS**, and **Vanilla JavaScript**.

Users are displayed as modern profile cards, and the search bar filters cards instantly while typing.

## ✨ Features

- 🔍 Real-time search filtering
- 🎴 Dynamic profile card generation
- 🖼️ Beautiful modern UI
- 🌈 Blurred background overlay effect
- 📱 Responsive layout
- ⚡ Built with Vanilla JavaScript (No Frameworks)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
├── index.html

├── script.js

```

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/user-search-cards.git
```

3. Run `index.html` in your browser.
